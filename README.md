# Aurum & Co. - Luxury Jewelry & Watches Store

A premium online e-commerce platform for luxury jewelry and men's watches with personalized engraving services.

## Features

- 🛍️ **Product Catalog**: Browse jewelry and watches with advanced filtering
- 💎 **Product Details**: Detailed product pages with specifications
- 🛒 **Shopping Cart**: Add/remove items, view cart summary
- 💳 **Checkout**: Secure checkout process with order summary
- 👤 **User Accounts**: Registration, login, order history
- ✍️ **Engraving Studio**: Personalized engraving with font selection
- 🔍 **Search & Filter**: Filter by material, price, collection
- 💰 **Payment Processing**: Integrated payment gateway
- 📦 **Inventory Management**: Real-time stock tracking

## Tech Stack

**Frontend:**
- React.js
- CSS3 with custom styling
- React Router for navigation
- Axios for API calls

**Backend:**
- Node.js with Express.js
- MongoDB for database
- JWT for authentication
- Stripe for payments

**Deployment:**
- Frontend: Vercel/Netlify
- Backend: Heroku/AWS
